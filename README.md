## Building

The only requirement is Go >= 1.21

```bash
git clone https://gitlab.com/nikkehtine/passgen
cd passgen
# You can run the program directly...
go run .
# ... or build it
go build .
./passgen # and whatever extension is at the end, if there is any
```
